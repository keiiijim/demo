GPT Ad Refresh — 3種類の実装デモ

Google Ad Manager (GAM) の GPT refresh() API を使った広告リフレッシュ実装比較

① TIME-BASED
時間ベース
設定秒数が経過したら自動でrefresh()を呼ぶ。最もシンプルだが視認性は考慮されない。
setTimeout視認性考慮なしActiveView低下リスク
② USER-ACTION
ユーザーアクション
スクロール・クリック・タブ切替などユーザー操作を検知してrefreshを発動。Googleが推奨。
impressionViewable視認率向上Googleが推奨
③ EVENT-BASED
イベントベース
スコア更新など、コンテンツ変化に連動してrefreshを発動。ライブ系コンテンツに最適。
content_update自然な体験スポーツ/速報
